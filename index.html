Селекторы бывают:

Селекторы по тегу (по названию элемента)
В качестве селектора может выступать любой элемент html, для которого определяются правила форматирования,такие как: цвет, фон, размер и др. p { text-align: justify; /* Выравнивание по ширине */ color: green; /* Зелёный цвет текста */ }
Cелекторы по id
Идентификатор определяет уникальное имя элемента, которое используется для изменения его стиля и обращения к нему через скрипты. #<Имя идентификатора> { Описание правил стиля }
Селекторы по классу
Классы применяют, когда необходимо определить стиль одного или нескольких элементов веб-страницы. Элементы при этом могут быть разными,но иметь частично или полностью похожие элементы оформления. В коде HTML имя класса определяется через атрибут class
Селекторы по нескольким классам
К любому элементу одновременно можно добавить несколько классов, перечисляя их в атрибуте class через пробел. В этом случае к элементу применяется стиль, описанный в правилах для каждого класса .class1.class2 { Описание правил стиля }.
Вложенные селекторы
При создании веб-страницы часто приходится вкладывать одни элементы внутрь других. Чтобы стили для этих элементов использовались корректно, помогут вложенные селекторы. Например, задать стиль для b только когда он располагается внутри контейнера p. Таким образом можно одновременно установить стиль для отдельного элемента, а также для элемента, который находится внутри другого. E F { Описание правил стиля } Здесь E это родительский элемент, а F — дочерний элемент, расположенный в контейнере E.

Так же есть такие селекторы: дочерние, соседние,селекторы по атрибуту. Используются гораздо реже
